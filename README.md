# P2P Ride Marketplace MVP

This app allows riders to post ride bids and drivers to accept or counter,
allowing the market to set the price.

## Core Flow
- Rider posts destination and bid
- Drivers accept or counteroffer
- Users chat and agree
- Ride is completed
- Both parties rate each other

## Target User
Drivers first. Riders second.

## Monetization
$5/month subscription for drivers (free during beta)
